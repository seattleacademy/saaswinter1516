brainfplusplus

Recently, I wrote an interpreter for the Brainf**k language, a simple esoteric array-based language Create by Swiss Physics student Urban Mueller. It was written in C++, and it will run any brainf**k program (No Guarantees as to how fast). Brainf**k is a Turing complete Language.

CsCalc

Another Project I have written recently is a windows forms calculator which can parse and solve mathematical expressions. It is written in C#, and I wrote in order to start learning the languae. I have some goals for it in the future, I think it would be cool to add a graphing aspect to it, and support for more advanced equation parsing.
